# Simple markdown editor

Just an experiment project to learn **WebAssembly** with **Golang**.

![Screen shot](/go-wasm-markdown-ss.png?raw=true)

### Run
- compile and run
```bash
$ ./compile_n_run.sh
```
- [localhost:8080](http://localhost:8080)
